# Task: Review Handoff Prep

## Goal
Подготовить пакет для независимого ревью после завершения разработки.

## Inputs
- Текущий commit/branch
- `framework/docs/definition-of-done-ru.md`
- `review/review-brief.md`

## Outputs
- `review/handoff.md`
- `review/test-results.md` (если прогонялись тесты)

## Rules
- Не менять код.
- Зафиксировать команды и результаты прогонов.
- Обозначить зоны повышенного риска.

## Done When
- Handoff заполнен, тестовые результаты приложены или явно отмечено, что тесты не выполнялись.
