# Review

Папка для независимого ревью и тестирования проекта. Здесь хранятся артефакты,
которые может прочитать и выполнить другой агент без контекста.

## Содержимое
- review-brief.md — контекст ревью, scope, цели, ограничения
- test-plan.md — план тестирования (по ТЗ/DoD)
- qa-coverage.md — факт выполнения тестов и покрытие
- code-review-report.md — находки и риски по коду
- bug-report.md — баги с воспроизводимостью
- runbook.md — пошаговая инструкция для независимого ревью

## Как использовать
1) Заполнить `review-brief.md` (commit/branch, окружение, команды тестов).
2) Сформировать `test-plan.md` параллельно разработке.
3) После завершения разработки выполнить ревью и тесты, заполнить остальные файлы.
4) Передать результаты основному агенту/команде.
