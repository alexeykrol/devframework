# Release Checklist

1) Все E2E тесты прошли.
2) Независимый review выполнен (см. `review/`).
3) Handoff для ревью подготовлен (`review/handoff.md`).
4) Проверен critical flow: Wizard → Results → Report → PDF.
5) Проверены auth и подписки (mock/real).
6) Проверены уведомления email/web‑push (mock/real).
7) Миграции БД применены.
8) Проверены RLS политики.
9) Обновлены legal‑страницы.
10) Проверена доступность (WCAG базовые требования).
11) Готов rollback план.
