# Definition of Done (DoD)

## Для фичи
- Реализована логика и UI.
- Покрыта unit‑тестами (если логика критическая).
- E2E сценарий обновлён/добавлен (если влияет на флоу).
- Нет ошибок/варнингов в консоли.
- Документация обновлена (если есть изменения требований).

## Для релиза
- Все E2E тесты проходят.
- Все critical flows проверены вручную.
- Независимый review выполнен (отчёт в `review/`).
- Handoff для ревью подготовлен (`review/handoff.md`).
- Миграции БД применены.
- Rollback план подготовлен.
- Обновлены release notes.
